# Nanocloud

## Install

###Install Ansible

On mac OS

```
brew install ansible
```


On GNU/Linux

```
sudo apt-get install ansible
```


###Install dependencies

```
ansible-galaxy install franklinkim.docker
ansible-galaxy install franklinkim.docker-compose
```

## Configure

## Run

Run Playbook

```
ansible-playbook nanocloud.yml
```
